Mudar Wallpaper

Posted by rap1 on November 18, 2023

NIJIKA. Script escrito em C

O objetivo de NIJIKA é de alterar o wallpaper baseado em um diretório que é indicado pelo utilizador, fazendo uma listagem das imagens dentro dele e dando a opção de escolher uma colocando ela como wallpaper, mesmo se forem colocadas novas imagens ele atualizará os números de escolha em outra sessão.

Uso de NIJIKA

  • É recomendado o uso do powershell

Primeiro clone o repositório do github para uma pasta (Menos diretamente nos Documentos).

git clone https://github.com/srcrapi/NIJIKA.git

Depois de clonar o repositorio terá de entrar na pasta NIJIKA depois na pasta bin

cd .\NIJIKA\bin

Execute o NIJIKA.exe assim que estiver na pasta bin

.\NIJIKA.exe

E deverá ver algum input que pede a localização da pasta dos wallpapers, vai ser preciso inserir a caminho completo da pasta onde contem todas as imagens que deseja como wallpaper.

Erros NIJIKA

Os erros que podem acontecer com NIJIKA é quando vamos colocar o caminho do diretório onde estão os wallpapers e nos enganos sem querer ou por colocar o caminho errado como está abaixo.

O erro acima foi porque não foi colocado o "Users" antes do username que é uma parte essencial.

Como resolver ?

Podemos resolver este problema de uma maneira muito simples, abra o explorer vá na pasta Documentos depois em NIJIKA e apague o arquivo dentro dela ou altere colocando o caminho corretamente

Conclusão

O programa é focado para o público que gosta de alterar o wallpaper constantemente (ou que têm vários) e querem alterar de uma vez e ver qual eles vão querer usar desta vez sem precisarem clicar sempre com o botão direito na imagem e selecionar como fundo com NIJIKA basta colocar o número corresponde e o wallpaper irá ser alterado.
