O objetivo de ryo é de dar a varias opções de aplicativos para usuário poder escolher e fazer uma instalação rápida e fácial dos aplicativos que ele mais ama.

Usage

É recomendado o uso do powershell

Primeiro clone o repositório do github para uma pasta.

$ git clone https://github.com/srcrapi/ryo.git

Depois de clonar o repositorio terá de entrar na pasta ryo depois na pasta src 

$ cd .\ryo\src

Execute o ryo.exe depois de ter instalado

$ .\ryo.exe
